# Data Analytics Project Portfolio

This repository contains two end-to-end data analytics projects focused on exploratory data analysis (EDA), data cleaning, and visualization using Python.

## Projects Included

### Airbnb Hotel Booking Analysis
- Analyzed booking patterns, pricing behavior, and customer trends to extract business insights.
- Performed data cleaning, exploratory data analysis, and visualization using Python libraries.

### Netflix Trend Analysis
- Explored content distribution, genre trends, and release patterns to identify platform-level insights.
- Applied EDA techniques and created visualizations to uncover trends.

## Tech Stack
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Jupyter Notebook  

## Key Skills Demonstrated
- Exploratory Data Analysis (EDA)  
- Data Cleaning  
- Data Visualization  
- Insight Generation  
- Business-Oriented Analysis
